# Bienvenido a la base de conocimiento común de ISPP
Esta es la base de conocimiento común de ISPP, donde se recopila información relevante aprendida durante el curso. Aquí se recogerán conocimientos generales y el feedback recibido durante el curso. 

El objetivo de esta base de conocimiento es proporcionar un recurso centralizado para que los estudiantes puedan acceder a información útil y relevante relacionada con el curso, durante el mismo y durante su futuro profesional.
