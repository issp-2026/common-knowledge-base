# Usuarios Pilotos

En esta página se recopila el feedback proporcionado por los profesores a los distintos grupos, con el objetivo de reunir una guía de buenas prácticas que ayude a mejorar la búsqueda de usuarios pilotos.

## Feedback General
- Conseguir un mínimo de 10–12 usuarios piloto comprometidos (idealmente 12–20).
- Incluir en la presentación una lista detallada de usuarios piloto y su tipo.
- Buscar pilotos para cada perfil de usuario.
- Formalizar un compromiso real de testeo.
- Diseñar un plan estructurado de gestión de pilotos:
    - Captación.
    - Calendario de pruebas.
    - Canales de comunicación.
    - Registro de lo mostrado (mockups/demos).
    - Registro del feedback.
    - Confirmación explícita sobre disposición de pago.
- Utilizar a los pilotos para validar el precio preguntando cuánto pagarían.
- Priorizar comunicación en vivo (reuniones / llamadas).
- Definir y explicar claramente el proceso de recogida de feedback.
- Ampliar progresivamente la base de usuarios piloto.
